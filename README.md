# 📌 ESP32 Temperature and Pressure Monitoring System

## 📖 Description
This project uses an ESP32 microcontroller to measure environmental temperature and pressure.

Initially, the project was designed as a wearable monitoring system, but due to hardware constraints, it was implemented as a stationary environmental monitoring system.

---

## ⚙️ Components Used
- ESP32  
- Pressure & Temperature Sensor (BMP280 / BME280)  
- Jumper Wires  
- Breadboard
- OLED  

---

## 🔧 Working Principle
The sensor collects environmental data:
- Temperature (°C)
- Pressure (hPa)

ESP32 reads this data and displays it via:
- Serial Monitor / Display / OLED
